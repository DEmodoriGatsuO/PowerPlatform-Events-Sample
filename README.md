# Power Platform 登壇イベント アプリのサンプル

## 概要
このリポジトリでは、イベントに利用したPower AppsやPower Automateのサンプルなど、公開出来る範囲で公開致します。
<br>
また、アプリで使うメディアは、基本的に[いらすとや](https://www.irasutoya.com/)のものを利用します。
<br>

## 構成
- README.md
<br>
イベントごとにフォルダを分け、アプリや資料を格納しております。
